# escapegromtheEnemy
Проста аркадна гра де синє коло тікає від червоного. 
Червоне коло постійно переслідує синє і з кожним дотиком стає швидшим. 
Коли ворог доганяє гравця, синє коло телепортується в випадкове місце на екрані.
Швидкість ворога збільшується на 5 одиниць після кожного дотику, але обмежена максимальним значенням 500. 
При кожному дотику лунає звуковий ефект монети.
Потрібно ще дописати фрагмент коду щоб синє коло не зупинялося при доходу до краю карти.
